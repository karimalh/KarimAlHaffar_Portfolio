# Assets Directory

This directory contains static assets for the portfolio website.

## Resume

Place the resume PDF file as `Fall_2025.pdf` in this directory. The website expects to find the resume at `/assets/Fall_2025.pdf`.

## Instructions for updating resume:

1. Replace the `Fall_2025.pdf` file with your updated resume
2. Make sure the filename remains `Fall_2025.pdf` or update the link in the Resume component
3. The file will be available for download via the "Download Resume" button on the website

## Project Images

When adding project images:
1. Add images in common web formats (jpg, png, webp)
2. Use descriptive filenames
3. Update the ProjectCard components with the correct image paths
4. Optimize images for web (recommended max width: 800px for project thumbnails)
